CSS Spinning Square Demo

This is a spinning square that turns into a circle.

This square spins by using only keyframes in CSS.


    .square {
        width: 100px;
        height: 100px;
        background-color: #333;
        margin: 45px auto;
    }
    .spin {
        animation: spinning 2.1s;
        animation-iteration-count: infinite;
        animation-direction: infinite;
        animation-timing-function: ease;
    }
    @keyframes spinning {
        from {
            transform: rotate(0deg);
            border-radius: 2px;
        }
        50% {
            transform: rotate(180deg);
            border-radius: 50px;
        }
        to {
            transform: rotate(0deg);
            border-radius: 2px;
        }
    }

CSS Stretching Square Demo

This stretches a square into a rectangle.

This square stretches into a rectangle using only keyframes in CSS.


    .square {
        width: 100px;
        height: 100px;
        background-color: #333;
        margin: 45px auto;
    }
    .stretch {
        animation: stretch 2.1s;
        animation-iteration-count: infinite;
        animation-timing-function: ease;
    }
    @keyframes stretch {
        0%,
        100% {
            transform: scale(1, 1);
        }
        50% {
            transform: scale(5, 0.05);
        }
    }

Hamburger to Arrow Transition

Hamburger menu button turns into an arrow.

Changes shape with fairly basic css transitions and translations. 


    #main-menu__open .main-menu__icon-item {
        background: #303133;
        width: 100px;
        height: 2px;
        margin-bottom: 27px;
        transform-origin: center;
        overflow: visible;
        position: relative;
        z-index: 1;
        display:block;
    }

    #main-menu__open .main-menu__open_one,
    #main-menu__open .main-menu__open_three {
        -webkit-transition: all .35s cubic-bezier(.1, .885, .32, 1.2);
        -ms-transition: all .35s cubic-bezier(.1, .885, .32, 1.2);
        transition: all .35s cubic-bezier(.1, .885, .32, 1.2);
        -webkit-transform: translateY(0);
        -ms-transform: translateY(0);
        transform: translateY(0);
    }

    #main-menu__open .main-menu__open_two {
        -webkit-transition: all .2s cubic-bezier(.1, .885, .32, 1.2);
        -ms-transition: all .2s cubic-bezier(.1, .885, .32, 1.2);
        transition: all .2s cubic-bezier(.1, .885, .32, 1.2);
    }

    #main-menu__open:active .main-menu__open_one,
    #main-menu__open:hover .main-menu__open_one {
        -webkit-transform: rotate(-45deg)scaleX(.8)translateX(-12px)translateY(-10px);
        -ms-transform: rotate(-45deg)scaleX(.8)translateX(-12px)translateY(-10px);
        transform: rotate(-45deg)scaleX(.8)translateX(-12px)translateY(-10px);
    }

    #main-menu__open:active .main-menu__open_two,
    #main-menu__open:hover .main-menu__open_two {
        -webkit-transform: translateX(7px);
        -ms-transform: translateX(7px);

    }

    #main-menu__open:active .main-menu__open_three,
